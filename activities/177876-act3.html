INTERPRETACIÓN DE POLÍTICAS DE FILTRADO (IPTABLES)

ANÁLISIS DE FLUJO DE PAQUETES Y TRADUCCIÓN DE REGLAS

/ETC/NET/PACKET_FLOW.LOG

1. Flujo del Paquete

Cuando un paquete llega al sistema...
1. Pasa por una Table (Tabla)
2. Después por una Chain (Cadena)
3. Finalmente se ejecuta una Rule/Action (Regla/Acción)

2. Propósito de las Tablas

TABLA PROPÓSITO PRINCIPAL EJEMPLO DE USO
FILTER Permite filtrar paquetes Bloqueo de tráfico
NAT Traducción de direcciones Port Forwarding
MANGLE Auditoría de Paquetes Denegar Ataques
RAW Seguimiento de Paquetes Trackear paquetes
SECURITY Analizar Propósito/Servicio Autorizar Servicios
/BIN/CMD_TRANSLATOR.SH

3. Anatomía de un Comando

iptables -A INPUT -p tcp -m multiport --dports 80,443 -j ACCEPT
4. Este comando permite:
Permite las conexiones HTTP / HTTPS.
5. Variables y Opciones Comunes:
  • a) Limitar intentos por minuto:
    --limit 5/minute
  • b) Filtrar por IP de origen:
    -s 192.168.1.0/24
  • c) Ver solo números, sin DNS:
    --list -n
  • d) Ver reglas con contadores (paquetes y bytes):
    iptables -L -v -n --line-numbers
6. ¿Qué hace esta regla compleja?
iptables -A INPUT -i eth0 -p tcp -m multiport --dports 22,80,443 \ -m state --state NEW,ESTABLISHED -j ACCEPT
Permite el tráfico TCP de los puertos 22, 80 y 443 en la interfaz eth0, revisando el estado de conexión (NUEVA o ESTABLECIDA).

Ejercicios de Escritura de Reglas

7. Permitir tráfico HTTP entrante:
iptables -A INPUT -p tcp --dport 80 -j ACCEPT
8. Permitir todo el tráfico saliente:
iptables -A OUTPUT -o wlan0 -j ACCEPT
9. Permitir SSH solo desde la IP 192.168.1.50:
iptables -A INPUT -p tcp -s 192.168.1.50 --dport 22 -j ACCEPT
10. Permitir tráfico TCP entrante a puertos 80 y 443 solo si es conexión establecida o relacionada:
iptables -A INPUT -p tcp -m multiport --dports 80,443 -m state --state ESTABLISHED,RELATED -j ACCEPT
11. Permitir tráfico TCP entrante por eth0 a 22, 80 y 443, registrar intentos y permitir solo NEW y ESTABLISHED:
iptables -A INPUT -i eth0 -p tcp -m multiport \ --log-prefix "intentos" --dports 22,80,443 \ -m state --state NEW,ESTABLISHED -j ACCEPT
